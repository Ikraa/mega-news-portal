Question:Difference between var , let and const?

Ans:The main difference between var , let and const is declaring the variable using var we can redeclare and reassign it . If we use let to declare a variable then we can reassign it but we can’t redeclare. On the other hand declare the variable using const then we can’t Redeclaring and reassign it .
